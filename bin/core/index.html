      Welcome to HIDE

      Start a new project or open an existing one.

      Create New Project... Open Project...

      Color Preview

      Click on color code in any file to activate color preview.
      Supported color formats: 0xFFFFFF, #FFFFFF.

      Color Picker

      Click on color preview to show color picker.
      Color will be applied if you click outside color picker, you can cancel color change if you press Escape, while color picker is shown.

      Open File Dialog

      Press Ctrl-Shift-O (or Cmd-Shift-O on Mac) in code editor to open file in new tab.

      Show Class List

      Press Ctrl-Shift-P (or Cmd-Shift-P on Mac) in code editor to show full list of classes available for current project.

      EReg Preview

      Click on regular expression in any file to see preview of it applied to current document.
